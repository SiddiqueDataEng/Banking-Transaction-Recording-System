# Banking Transaction Recording System

****

## Overview
Customer transactions with audit controls

## Technology Stack (2013)
- **Tools**: MSSQL, Triggers
- **Database**: BankingDB


## Features
- Production-ready database schema
- Stored procedures and triggers
- Sample data and test scripts
- Complete documentation

## Installation
1. Execute database/schema.sql
2. Review database/procedures.sql (if exists)
3. Load database/seed-data.sql (if exists)
4. Configure application connection strings
